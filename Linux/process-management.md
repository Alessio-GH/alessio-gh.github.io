# Process management

## - Show running processes 🏃
```bash
$ ps -ef
```
> [!TIP]
> Better use | more or just *ps*.


## - [Real time running processes view 👀](running-processes.html)
```bash
$ top
```

## - [Kill processes 💀](kill-processes.html)
```bash
$ kill
```

## - [Background execution 🌤️](bg-processes.html)
```bash
$ bg
```
